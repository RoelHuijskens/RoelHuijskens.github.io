Tic tac toe

After watching the very intriguing documentary about AlphaGo, a program created by DeepMind, I got interested in the field of reinforcement learning. I am currently (slowly) working through a book on reinforcement learning by Aske Plaat. This little program is my very first implementation of the ideas about reinforcement learning presented in the third chapter. It consists of a recursive function which will play out all of the possible games of tic tac toe on a 3x3 playing field and will decide based on all the outcomes down the line (win (+0.99), lose(-1) or tie(0)) ) which move to pick next against the current opponent.
