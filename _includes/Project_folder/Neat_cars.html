Self driving cars with NEAT

This program aims to learn how to steer a car around a given track using a 1 hidden-layer feed-forward neural network. This network is optimized by creating 50 different network configurations each generation and assigning each network to a car. The input variables are the distances between the sides of the track and the car based on the 3 different vision traces presented in green in the example on the right, the output variables determine whether the car will break (the gas paddle is always fully pressed) and whether the car should steer left or right. Next depending on how well the car performs (the more checkpoints it passes) the more cars will resemble that configuration in the next generation. The game and scoring mechanics are all made by me, the NEAT module is utilized for creating different generations of neural networks.
