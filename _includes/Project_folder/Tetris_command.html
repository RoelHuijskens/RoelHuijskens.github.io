Tetris in the command prompt

This project was one of my first serious steps into python, the idea is based on a youtube video by jarvidx9 a youtube channel I would highly recommend you check out. However, the idea presented in the video is executed in C++ and at the time of starting this project, I had no experience in C++ whatsoever. However, I was intrigued by the idea and decided to have a go at it myself only using the general idea introduced by jarvidx9 and implementing it in python. The main drawback of this interpretation over the idea presented in the video is the fact that python has (to my knowledge) no module which allows for the refresh rate of the command prompt to synchronize with the update of the game. If such a module does exist, please let me know.

The way this game is played is very simple, call the python script in a directory of your choosing and fill in your name. Next, use the a key to move the currently active block to the left, d to move it to the right, and w to turn the block clockwise. Normal Tetris rules apply and the scores are based on the standard Tetris rules. The better you play, the faster the blocks fall and the higher the score awarded per block

Good luck
